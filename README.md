# Autoworks - v0.1
## Commands
```
help - return list of commands  
yo mama - returns a random yo mama joke from autoworks api  
dad joke - returns a random dad joke from autoworks api  
cat fact - returns a random cat fact from autoworks api  
task - returns a random task for you to do ("go touch grass" not included, sorry gamers)  
joke - returns a joke from a joke api  
time - returns local time from the hosts computer (sorry timezones, your not included)  
hello - says hello, an example of using the variables returned by getMessage()  
cool - returns a random percentage of how cool you are. another example of using the getMessage() returned variables  
talking ben - have a consversation with talking ben, use the message body if you want but its not required  
```
